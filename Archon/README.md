# Archon

**Advanced Roblox Exploit Framework (Rage Base)**

Современный модульный чит с акцентом на качество и расширяемость.

## Основные возможности
- Чистая архитектура (DI + Scheduler + EventBus)
- Aimbot + Silent Aim
- Продвинутый Anti-Aim (Jitter, Spin, Desync, FakeLag и т.д.)
- Exploits (Fly, Noclip, Hitbox Expander, Teleport и др.)
- Гибкий конфиг с hot-reload

## Установка
1. Скачай папку `Archon`
2. Запусти `main.lua` через executor

**Проект в активной разработке.**
