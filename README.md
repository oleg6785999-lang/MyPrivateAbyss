# Archon

**Advanced Roblox Exploit Framework (Rage Base)**

Современный модульный чит с акцентом на производительность, расширяемость и rage-стиль.

## Основные возможности

- **Модульная архитектура** — DI + Scheduler + EventBus + чистые слои
- **Качественный Aimbot + Silent Aim**
- **Продвинутый Anti-Aim** (Jitter, Spin, Desync, Fake Lag, Freestanding, Manual)
- **Мощные Exploits** (Fly, Noclip, Hitbox Expander, CFrame Teleport, SpeedHack и др.)
- **Гибкая система конфигурации** с hot-reload
- **Чистый код** и удобная расширяемость

## Структура проекта

- `core/` — ядро фреймворка
- `services/` — сервисный слой (данные и состояние)
- `modules/` — основные модули (aimbot, antiaim, exploits и т.д.)
- `ui/` — интерфейс (Rayfield)
- `config/` — конфигурационные файлы

## Установка

1. Скачай репозиторий как ZIP
2. Распакуй папку `Archon`
3. Запусти `main.lua` через свой executor (Synapse X, Fluxus, Wave и т.д.)

## Предупреждение

Данный проект создан в **образовательных и исследовательских целях**.  
Использование читов нарушает правила Roblox и может привести к бану аккаунта.
