# Archon

Advanced Roblox Exploit Framework (Rage Base)

Современный модульный чит с акцентом на:
- Качественный Aimbot + Silent Aim
- Продвинутый Anti-Aim
- Мощные Exploits
- Чистую архитектуру (DI + Scheduler + EventBus)

## Особенности
- Полноценная модульная архитектура
- Поддержка Anti-Aim (Jitter, Spin, Desync, FakeLag и т.д.)
- Exploits (Fly, Noclip, Hitbox Expander, Teleport и др.)
- Конфиг с hot-reload

## Установка
1. Скачай репозиторий
2. Запусти `main.lua` через свой executor

## Структура
- `core/` — ядро фреймворка
- `modules/` — основные функции
- `services/` — данные и состояние
